Fullstack technologies

html

HyperText Markup Language (HTML) is a text-based language that defines the structure and content of web pages

css

CSS stands for Cascading Style Sheets, a computer language used to style and structure web pages. It's a key component of web development, along with HTML and JavaScript, and is responsible for the overall look of a website

java script

java script is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.


back-end:

Node js

Node.js is an open-source, cross-platform JavaScript runtime environment that allows developers to execute JavaScript code outside of a web browser.

python

Python is a high-level, interpreted, object-oriented programming language known for its simple, readable syntax, making it a popular choice for various software development tasks including web applications, data analysis, machine learning, and automation, often praised for its beginner-friendly nature and extensive standard library

React js

React js is a free and open-source front-end JavaScript library for building user interfaces based on components.
